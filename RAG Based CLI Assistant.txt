Getting code from my semantic search engine repo:

git remote add old-project https://github.com/SreeragS2003/Semantic-Search-Engine.git

2. Pull the code from the repo
git pull old-project main --allow-unrelated-histories

3. Push this code to new repo
git push origin main


Let us create requirements.txt.

1. Install the required packages first -

	pip install google-genai
	pip install python-dotenv

2. Add the current list of installed packages to requirements.txt file - 
	
	pip freeze > requirements.txt (has to be done every time new package is installed so that requirements.txt is updated)

Improving retrieval:

1. Semantic search (FAISS)
2. Keyword search (BM25-style)
3. Combine + rerank


Full pipeline summary after reranking added:

Offline:
PDF → chunks → embeddings → FAISS → saved

Online:
Query → embedding → FAISS → candidates
       → rerank → top chunks as context
       → query + context to LLM → answer


After including agentic AI,

LLM is called repeatedly
Each time with:
    same question
    + more context
    + more observations

Each call to the llm includes:

- instructions (AGENT_PROMPT)
- full reasoning trace (history)
- original query


Prompt flow:

CALL 1 — INITIAL DECISION
Prompt sent to LLM:
You are an intelligent agent...

Previous steps:


User: What is neural networks and calculate 25 * 17?
LLM response:
Thought: I need info and a calculation
Action: search_docs
Input: neural networks

Key point

First call = planning step

CALL 2 — AFTER FIRST TOOL

Now history is updated:

Thought: I need info and a calculation
Action: search_docs
Input: neural networks

Observation: (context from RAG)
Prompt becomes:
You are an intelligent agent...

Previous steps:
Thought: I need info and a calculation
Action: search_docs
Input: neural networks

Observation: Neural networks are...

User: What is neural networks and calculate 25 * 17?
LLM response:
Thought: Now I should calculate
Action: calculator
Input: 25 * 17

Key point

Second call = refined decision using tool output

CALL 3 — AFTER SECOND TOOL

History now:

Thought: ...
Action: search_docs
Observation: ...

Thought: ...
Action: calculator
Observation: 425
Prompt:
You are an intelligent agent...

Previous steps:
...full history...

User: What is neural networks and calculate 25 * 17?
LLM response:
Thought: I now have everything
Action: final
Answer: Neural networks are... Also 25 * 17 = 425

Key point

Final call = answer generation

Summary of calls
Call	Purpose
1	decide first action
2	refine after tool
3	final answer
