Semantic search:

[Offline phase]
Documents → Chunk (every line is a chunk in this project) → Embedding → Store in FAISS

[Online phase]
User Query → Embedding → Similarity Search (FAISS) → Top-k relevant chunks



FAISS index = in-memory vector database (vectors only) [stored inside RAM]

VectorStore = wrapper that:
    - holds FAISS index
    - holds original texts
    - connects both
